# BioNeMo Framework: Available Models

State-of-the-art models are continually integrated into the BioNeMo Framework. The BioNeMo Framework currently offers the following pre-trained models:

| **Model**                     | **Modality** | **Uses**                | **Training Location**                                                                    |
| ----------------------------- | ------------ | ----------------------- | ---------------------------------------------------------------------------------------- |
| [AMPLIFY](./amplify.md)       | Protein      | Representation Learning | [bionemo-recipes](https://github.com/NVIDIA/bionemo-framework/tree/main/bionemo-recipes) |
| [ESM-2](./ESM-2/index.md)     | Protein      | Representation Learning | [bionemo-recipes](https://github.com/NVIDIA/bionemo-framework/tree/main/bionemo-recipes) |
| [Evo2](./evo2.md)             | DNA          | Generative AI           | sub-packages (5D parallel)                                                               |
| [Geneformer](./geneformer.md) | Single Cell  | Representation Learning | [bionemo-recipes](https://github.com/NVIDIA/bionemo-framework/tree/main/bionemo-recipes) |

For more information about the models included in BioNeMo Framework, refer to the Model Cards linked in the table above or the original publications referenced in the respective model descriptions.
